# g0v-event

## 欄位
- name: 活動名稱
- id: 活動代碼（不重覆）
- time:
  - date: 活動日期
  - time_start: 活動開始時間
  - time_end: 活動結束時間
- location: 活動地點
  - name: 地點名稱
  - address: 地點地址
  - lat: 緯度
  - lng: 經度
- description: 活動描述
- link
  - event: 活動網址
  - collaborate: 協作網址
  - video_pitch: 提案影片網址
　- video_talk: 短講影片網址
  - video_demo: 展示影片網址
